# MedSpa Review Follow-Up — Feature Roadmap

## Core MVP
1. account login
2. review link settings
3. CSV/manual contact import
4. campaign creation
5. initial review request
6. one follow-up
7. basic activity dashboard

## Top Upgrades After Core MVP
### 1. Google rating + review count dashboard
Show:
- current Google rating
- total review count
- simple snapshot in the main dashboard

### 2. Review growth tracking over time
Show:
- review count trend
- rating trend if available
- before/after feel since using the product

### 3. Click tracking
Track:
- link clicks
- campaign-level click counts
- basic performance visibility

### 4. Med spa-specific message templates
Include:
- post-visit review request templates
- follow-up templates
- tone options that fit med spa branding

### 5. Simple performance reporting
Show:
- contacts imported
- requests sent
- follow-ups sent
- clicks
- campaign summary performance

## Later Features
- negative feedback routing
- multi-location support
- integrations
- agency mode
- review feed in dashboard
