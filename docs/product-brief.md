# MedSpa Review Follow-Up — Product Brief

## Working Name
MedSpa Review Follow-Up

## Positioning
A simple Google review request and follow-up system built specifically for med spas that want more 5-star reviews without paying for a bloated reputation platform.

## Core Promise
Help med spas get more Google reviews with a lightweight follow-up workflow that is easier and cheaper than all-in-one review software.

## Target Buyer
- med spa owners
- med spa managers
- med spa marketing operators
- agencies serving med spas

## Core Problem
Med spas know reviews matter for local trust and bookings, but many either:
- don’t ask consistently
- ask manually
- use generic workflows
- or pay for large reputation suites with far more features than they need

## Why This Niche
- reputation-sensitive
- premium feel matters
- local search matters
- review count and quality influence trust
- likely willing to pay for cleaner automation

## Wedge
Not a full reputation platform.
Not an inbox.
Not CRM.

Just:
- request review
- follow up once or twice
- direct patients/clients to Google review page
- keep the workflow simple

## Recommended Pricing
- founding plan target: $39/month
- possible early lower test: $29/month
- later pro/agency tiers if demand exists

## MVP Goal
Build the smallest useful version that can be demoed and sold to a med spa within 1–2 weeks.
